# 5 day Forecast app

## Getting Started

### Installing

```
npm install
```
or

```
yarn install
```

### Start
```
npm start
```
or 
```
yarn start
```

### Build
```
npm run build
```
or 
```
yarn build
```
## Testing

### Test
```
npm run test
```
or 
```
yarn test
```

### Test watch
For development purpose watch edited files

```
npm run test:watch
```
or
```
yarn test:watch
```

### Test Coverage files

```
npm run test:coverage
```
or
```
yarn test:coverage
```
